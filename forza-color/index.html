DxBang / forza-color

Convert any color into the paint values used in Forza Motorsport & Forza Horizon.

Color Input

Forza Paint Values (0 – 255)

Hue – 0 – 255
Saturation – 0 – 255
Luminosity – 0 – 255

Forza Motorsport and Forza Horizon store paint colors as H S L values each ranging from 0 to 255, mapped from the standard HSL color model (Hue 0–360°, Saturation 0–100%, Lightness 0–100%).
